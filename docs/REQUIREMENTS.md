## 1. Overview
**Test Wright** is a workflow integration layer designed to orchestrate test execution, persist results, and surface integration documentation within existing CI/CD pipelines. This tool addresses the validated pain point of fragmented test telemetry and onboarding friction in autonomous development workflows. It extends the Axentx portfolio by providing the orchestration layer complementary to low-level infrastructure (e.g., distinct from the iceoryx2 IPC library).
